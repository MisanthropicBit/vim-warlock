INTRODUCTION                                  *vim-warlock-intro* *warlock-intro*

A dark colorscheme with support for many languages, themes for vim-airline
and lightline.vim, and the following plugins.

    * Plugins
        - CtrlP
        - fzf.vim
        - majutsushi/tagbar
        - Netrw
        - scrooloose/nerdtree
        - vim-exchange
        - vim-gitgutter
        - vim-task
        - vimtex

==============================================================================
REQUIREMENTS                                         *vim-warlock-requirements*

This colorscheme has no requirements.

==============================================================================
CONFIGURATION                                     *vim-warlock-configuration*

Name                        Default     Description~
g:warlock#debug             0           Used for debugging the colorscheme.

g:warlock#bold              1           Enable all bold highlights.

g:warlcok#italic            1           Enable all italic highlights.

g:warlock#underline         1           Enable all underline highlights.

g:warlock#undercurl         1           Enable all undercurl highlights.

g:warlock#italic_strings    1           Highlight strings as italic.
